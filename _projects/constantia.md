`constantia` is my personal general 3D engine to learn graphics and game engine related stuff. It's named constantia cause i intent to be working on it constantly and not abandon it (long term).

This project is originated from my first project in 3D engine development named `valiance` that was very much copied/inspired from Hazel and i wanted to start fresh and relearn the basics.

This at the moment is in an early stage as i am using vulkan instead of OpenGL but after a lot of setup it has the capability to render one beautiful triangle.
