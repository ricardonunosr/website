`valiance` is a my first attempt at 3D engine from "scratch" with C++. This was
mostly copied/inspired by the youtuber
[TheCherno](https://www.youtube.com/c/TheChernoProject).

This was some basic 3D capabilities like textures, camera, instancing. Has some
integration with [ImGui](https://github.com/ocornut/imgui) and uses
[GLFW](https://github.com/glfw/glfw) for window creation/input,
[Glad](https://github.com/Dav1dde/glad) for loading OpenGL function pointers
from driver and [glm](https://github.com/g-truc/glm) for some complex math.
